BrowserNotif.JS

Lets a web page send notifications that are displayed outside the page at the system level

View on GitHub Download .zip Download .tar.gz

Instruction Demo Page

  1. Click here to request notification permission, click allow
  2. Notification will be appear
  3. Switch to another app, notification will be still visible
  4. You can click on the notification
  5. Open console and type 'notif1.close()' it will close the first notification pragmatically
